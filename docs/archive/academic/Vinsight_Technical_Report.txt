DETAILED TECHNICAL IMPLEMENTATION OF THE AGENT

THE THESIS AGENT IS AN ASYNCHRONOUS ORCHESTRATOR WITHIN THE INTELLIGENCE GRID, OPERATING ON A PLAN-RETRIEVE-REFLECT-REPORT ARCHITECTURE. UPON HYPOTHESIS SUBMISSION, THE AGENT GENERATES A TARGETED RESEARCH PLAN TO IDENTIFY INFORMATION GAPS. IT UTILIZES A PGVECTOR STORE TO SEMANTICALLY SEARCH SEC FILINGS AND NEWS ARCHIVES. THE CORE REASONING ENGINE USES CHAIN-OF-THOUGHT (COT) PROMPTING TO SIMULATE ADVERSARIAL PERSONAS. A CRITICAL COMPONENT IS THE "AMD BUFFER," WHICH HANDLES PARTIAL DATA BY DYNAMICALLY RECALCULATING WEIGHTS FOR REMAINING METRICS TO PREVENT NULL PROPAGATION ERRORS. EVERY ASSERTION IS SUBJECT TO AN EVIDENCE GROUNDING GUARDRAIL THAT PERFORMS SEMANTIC MATCHING; ASSERTIONS LACKING A 30 PERCENT WORD-MATCH DENSITY WITH RETRIEVED TEXT ARE TAGGED AS [UNVERIFIED] TO ELIMINATE HALLUCINATION RISK.

OUTPUT EVALUATION AND VALIDATION

VALUATION AND VALIDATION ARE MANAGED THROUGH A "KILL-SWITCH" ARCHITECTURE AND CFA COMPOSITE THEORY. THE EVALUATION SYSTEM APPLIES "THE ANALYST'S VETO" BEFORE ANY VERDICT IS RELEASED.
